# 碳排放报告生成器 - 项目结构说明

## 项目概述

这是一个基于 Python 的碳排放报告生成工具，使用 Jinja2 模板引擎动态生成 Word 格式的碳排放盘查报告。

## 核心文件

### 主程序
- **main.py** (19 KB)
  - 主程序入口
  - 负责数据提取、格式化和报告生成
  - 支持命令行参数：`--generate`

### 数据处理
- **data_reader.py** (121 KB)
  - 数据读取模块
  - 从 Excel/CSV 文件中提取碳排放数据
  - 实现 Flags 系统（条件渲染）
  - 提供量化方法说明

### 模板和过滤
- **template.docx** (861 KB)
  - Word 报告模板
  - 使用 Jinja2 标签实现动态内容
  - 支持条件渲染（基于 flags）

- **jinja2_filters.py** (5.9 KB)
  - Jinja2 自定义过滤器
  - 数字格式化、排放量显示等

### 数据文件
- **test_data.xlsx** (294 KB)
  - 测试数据（大冶特钢公司数据）
  - 包含范围一、二、三的排放数据

- **减排行动统计.csv** (7.4 KB)
  - 减排行动统计数据
  - 包含公司基本信息

### 配置文件
- **requirements.txt** (51 bytes)
  - Python 依赖包列表
  - openpyxl, python-docx, docxtpl, jinja2

- **README.md** (6.9 KB)
  - 项目说明文档

### 工具目录
- **tools/** (8个Python脚本)
  - 数据分析工具
  - 模板检查工具
  - 调试辅助脚本

- **templates/** (2个HTML文件)
  - HTML模板文件

## 使用方法

### 安装依赖
```bash
pip install -r requirements.txt
```

### 生成报告
```bash
python main.py --generate
```

生成的报告将保存为 `carbon_report.docx`

### 测试 Flags 系统
```python
from data_reader import ExcelDataReader

reader = ExcelDataReader('test_data.xlsx')
data = reader.extract_data_from_xlsx_dynamic()

# 检查 flags
flags = data.get('flags', {})
print(flags)
```

## 核心特性

### 1. Flags 系统
- 自动判断各范围是否有数据
- 控制章节显示/隐藏
- 支持范围三的15个类别单独判断

### 2. 动态模板
- 使用 Jinja2 标签实现条件渲染
- 支持循环遍历（范围三的15个类别）
- 自动格式化数字（千分位分隔符、小数位数）

### 3. 数据类型处理
- 正确处理字符串 "0"
- 自动转换为浮点数
- 兼容各种数据格式

## 输出文件

- **carbon_report.docx** - 最新生成的报告
  - 包含完整的碳排放盘查信息
  - 动态显示有数据的范围和类别
  - 格式化的数字和表格

## 技术栈

- Python 3.14+
- openpyxl - Excel文件读取
- python-docx - Word文档处理
- docxtpl - Word模板引擎（基于Jinja2）
- Jinja2 - 模板引擎

## 注意事项

1. 确保 Excel 数据文件格式正确
2. CSV 文件使用 UTF-8 或 GBK 编码
3. 模板文件必须保持 Jinja2 标签完整
4. 生成报告前检查 flags 系统状态
